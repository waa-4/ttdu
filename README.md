# Travel The Damn Universe v1

A simple browser idle/action game.

## Included

- Automatic cube movement
- Speed, money-per-second, and boulder-distance upgrades
- Boulder crush and run reset
- Rebirths with permanent +5% speed each
- 10 worlds with increasing money multipliers
- Click/tap shooting
- Zombie enemies and bonus money
- Random events
- Skins
- Temporary boosts
- Redeemable codes
- Local save data
- Supabase leaderboard support
- Responsive PC/mobile canvas

## Codes

- FIRSTSTEP
- BIGROCK
- CUBEPOWER

## Leaderboard setup

Open your Supabase project, go to SQL Editor, and run `SUPABASE-SETUP.sql` once.

The publishable URL and key are already included in `game.js`.

## Running

Open `index.html` directly, or upload the files to GitHub Pages.
